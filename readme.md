# mina-cli

## Get mina-cli on NPM
```
npm install -g minapp-cli
```

## Create minapp project with mina-cli
```
mina create <project's name>
```

## Check out your version of mina-cli
```
mina --version
```

## Seeking for help
```
mina --help
```
You can find it [here](https://www.npmjs.com/package/minapp-cli) at NPM.
